# Amarnath Yadav - Full Stack Developer Portfolio

A stunning, fully-responsive single-page portfolio website built with vanilla HTML, CSS, and JavaScript.

## 🎯 Features

✅ **Stunning Design**
- Dark theme with electric purple (#7C3AED) and neon cyan (#06B6D4) accents
- Bold, creative typography using Space Grotesk & Inter fonts
- Glassmorphism cards with backdrop blur effects
- Smooth animations and transitions throughout

✅ **Interactive Elements**
- Typing animation on hero heading (cycles through roles)
- Custom cursor effect with trailing ring (desktop only)
- Scroll-reveal animations for all sections
- Hover effects on cards, buttons, and links
- Fixed navigation with smooth scrolling

✅ **Responsive Design**
- Fully mobile-ready with hamburger menu
- Optimized for tablet and desktop
- Touch-friendly on all devices
- Adaptive grid layouts

✅ **Performance**
- Single HTML file (no build process needed)
- No external frameworks or heavy libraries
- Fast load time - ready for GitHub Pages & Vercel
- Clean, optimized CSS and vanilla JavaScript

## 🚀 Quick Setup

### Step 1: Prepare Profile Image
1. Take the profile image (profile.jpg) and save it in the same folder as `index.html`
2. Name it exactly: `profile.jpg`
3. Recommended: Square image, at least 400x400px

### Step 2: Local Testing
Simply open `index.html` in any modern browser:
```bash
# Windows - In the Practice folder
start index.html

# Or double-click index.html in Explorer
```

### Step 3: Deploy to GitHub Pages
1. Create a GitHub repository (name it anything, e.g., `portfolio`)
2. Push these files to the repository:
   - `index.html`
   - `profile.jpg`
3. Go to Repository Settings → Pages
4. Select "Deploy from branch" → main branch → root folder
5. Your site will be live at: `https://yourusername.github.io/portfolio`

### Step 4: Deploy to Vercel
1. Sign up at [vercel.com](https://vercel.com)
2. Click "New Project"
3. Import your GitHub repository
4. Click "Deploy"
5. Your site will be live at: `https://your-project-name.vercel.app`
