```bash

```

### Step 3: Customize (Optional but Recommended)
- Update your name, email, GitHub, LinkedIn in index.html
- Update project details
- Update experience details
- Update skill list

### Step 4: Deploy to GitHub
```bash
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
git push -u origin main
```

### Step 5: Enable GitHub Pages
- Go to repository Settings
- Select Pages
- Choose: Deploy from branch → main → root
- Live at: https://YOUR_USERNAME.github.io/portfolio

### Step 6: (Optional) Deploy to Vercel
- Sign up at vercel.com
- Click "New Project"
- Import GitHub repository
- Click "Deploy"
